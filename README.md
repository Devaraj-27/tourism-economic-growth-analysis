# Tourism Economic Growth Analysis

## Overview

This project analyzes the relationship between tourism development and economic growth across 15 Indian states from 2011–2023 using panel data econometric techniques.

## Objectives

- Examine the impact of domestic tourism on economic growth
- Analyze the contribution of foreign tourism
- Compare fixed effects and random effects estimators

## Methodology

The study applies panel data econometric methods including:

- Fixed Effects Model
- Random Effects Model
- Hausman Test
- Driscoll-Kraay Standard Errors

## Tools Used

- Stata
- Panel Data Econometrics
- Statistical Analysis
- Excel

## Repository Structure

```text

```

## Key Findings

- Domestic tourism positively influenced economic growth
- Foreign tourism showed statistically limited impact
- Fixed effects estimation provided more reliable results
